Polymeria

Polymeria is a set of Polymer Web components for rendering an Eclipse UML2 model file. Because of custom elements compatibility constraints (dash needed), the model file cannot be rendered without a transformation of UML element names.

To test, just clone the repository and have a look at the sample.html file.
